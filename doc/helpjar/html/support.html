Online support
Du kan med den här funktionen öppna en webbläsare för att gå till hemsidan för onlinesupport där du kan få support för din produkt. För att kunna använda supporten så måste du registrera din produkt, det kan antingen göras via installations programmet eller via supporthemsidan. Detta är samma sida som för uppdateringen så du behöver bara registrera din en gång.

Skulle inte hemsidan öppnas så kan du manuellt gå till supportsidan via länken https://www.fribok.org/
