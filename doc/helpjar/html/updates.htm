Online uppdatering
Ni kan med den här funktionen öppna en webbläsare för att gå till hemsidan för onlineuppdatering där ni kan uppdatera er produkt. För att kunna uppdatera er produkt så måste ni registrera den, det kan antingen göras via installations programmet eller via supporthemsidan. Detta är samma sida som för supporten så ni behöver bara registrera er en gång.

Skulle inte hemsidan öppnas så kan ni manuellt gå till supportsidan via länken http://support.jfsbokforing.se
