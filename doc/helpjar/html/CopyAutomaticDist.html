Kopiera automatfördelning
I det här formuläret kan ni ändra i en kopierad automatfördelning.

Följande fält finns när ni skall ändra en kopierad automatfördelning:

Konto
Här skriver ni in eller väljer ett kontonummer för automatfördelningen.

Beskrivning
Här skriver ni in en text för att beskriva automatfördelningen.

Belopp
Här kan ni skriva in ett belopp som skall bokföras på det ovan valda kontot för automatfördelningen. Om ni anger ett positivt belopp så kommer beloppet att bokföras i debet och om ni anger ett negativt belopp så kommer beloppet att bokföras i kredit.

Konto
Här skriver ni in eller väljer ett kontonummer för konteringsraden.

Beskrivning
Här visas en text som beskriver kontot och mer specifikt anger vilka affärshändelser som skall bokföras på kontot. Beskrivningen hämtas från kontoplanen enligt valt kontonummer enligt ovan

Procent
Här skriver ni in ett procentuellt värde (ange ej %) som kommer att bokföra ett belopp på kontoraden som motsvarar det här värdets procentuella andel av värdet på kontot för automatfördelningen. Om ni anger ett positivt värde så kommer beloppet bokföras i debet och om ni anger ett negativt värde så kommer beloppet bokföras i kredit.

Debet
Här skriver ni in ett monetärt belopp som kommer att bokföras i debet på konteringsraden.

Kredit
Här skriver ni in ett monetärt belopp som kommer att bokföras i kredit på konteringsraden.

Projekt
Här kan ni välja eller skriva in ett projektnummer för att specificera intäkten eller kostnaden på ett visst projekt.

Resultatenhet
Här kan ni välja eller skriva in ett resultatenhetsnummer för att specificera intäkten eller kostnaden på en viss resultatenhet.
