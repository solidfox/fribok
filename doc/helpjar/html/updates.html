Online uppdatering
Du kan med den här funktionen öppna en webbläsare för att gå till hemsidan för onlineuppdatering där du kan uppdatera din produkt. För att kunna uppdatera din produkt så måste du registrera den, det kan antingen göras via installations programmet eller via supporthemsidan. Detta är samma sida som för supporten så du behöver bara registrera din en gång.

Skulle inte hemsidan öppnas så kan du manuellt gå till supportsidan via länken https://www.fribok.org/
