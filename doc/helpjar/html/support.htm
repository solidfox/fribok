Online support
Ni kan med den här funktionen öppna en webbläsare för att gå till hemsidan för onlinesupport där ni kan få support för er produkt. För att kunna använda supporten så måste ni registrera er produkt, det kan antingen göras via installations programmet eller via supporthemsidan. Detta är samma sida som för uppdateringen så ni behöver bara registrera er en gång.

Skulle inte hemsidan öppnas så kan ni manuellt gå till supportsidan via länken http://support.jfsbokforing.se
