Importera verifikationer
Med den här funktionen kan ni importera verifikationer i filformatet SIE4 som lägger sig efter redan registrerade verifikationer. Det här kan vara aktuellt vid överföring av bokföringsunderlag från exempelvis ett fristående löneprogram, faktureringsprogram eller från revisorn/redovisningskonsulten som har skapat verifikationer till er.

Ni kan inte lägga till verifikationer som innehåller konton vilka inte finns i kontoplanen för aktivt bokföringsår, ni måste därmed lägga till alla konton som finns i SIE4-filen i den aktiva kontoplanen för bokföringsåret, det vill säga via "Register" och "Ändra kontoplan för aktivt år".
